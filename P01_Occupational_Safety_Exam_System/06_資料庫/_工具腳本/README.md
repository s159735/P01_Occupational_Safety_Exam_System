# 📁 資料庫工具腳本 (v10.1.0)

## 📋 簡介
本目錄包含所有與 Supabase 資料庫操作相關的工具腳本，支援 **8 大題型完整格式**。

## 🛠️ 腳本列表

| 腳本 | 功能 | 使用方式 |
|------|------|---------|
| `01_連線測試.py` | 測試 Supabase 連線 | `python 01_連線測試.py` |
| `02_建立資料表.py` | 建立 v10.1.0 完整資料表結構 | `python 02_建立資料表.py` |
| `03_匯入題目.py` | 匯入 8 大題型 | `python 03_匯入題目.py` |
| `04_匯入法規.py` | 匯入完整法規資料 | `python 04_匯入法規.py` |
| `05_驗證資料.py` | 驗證 v10.1.0 匯入結果 | `python 05_驗證資料.py` |

## 📂 資料來源路徑

| 資料類型 | 路徑 |
|----------|------|
| 題庫 (8 大題型) | `06_資料庫/02_種子範本/` |
| 法規資料庫 | `06_資料庫/07_法規資料庫/` |

## 🚀 執行順序

```bash
# 1. 先測試連線
python 01_連線測試.py

# 2. 建立資料表（若無）
python 02_建立資料表.py

# 3. 匯入題目（8 大題型）
python 03_匯入題目.py

# 4. 匯入法規
python 04_匯入法規.py

# 5. 驗證結果
python 05_驗證資料.py
