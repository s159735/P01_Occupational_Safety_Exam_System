【一】題庫欄位總覽（完整版）

欄位名稱                     型別         必填  說明
─────────────────────────────────────────────────────────────────────────
id                           字串         是    題目編號（前綴+數字）
type                         字串         是    題型代碼
typeLabel                    字串         是    題型中文名稱（完整繁體）

text                         字串         是    題目文字內容
options                      陣列         視題型 選項列表
answer                       數字/陣列/字串 是  正確答案（格式依題型）
points                       字串         是    總配分（如 "10%"）

law.pcode                    字串         是    法規代碼（首要鍵值）
law.name                     字串         建議    法規名稱（次要鍵值）
law.article                  物件         建議    條文定位
law.article.full             字串         建議    完整引用文字
law.article.條               字串         建議    條號
law.article.項               字串         建議    第幾項
law.article.款               字串         建議    款號
law.article.目               字串         建議    目號
law.summary                  字串         建議    法規摘要
blanks                       陣列         填空題  填空標記
formulaKey                   字串         計算題  公式代號
formulaParams                物件         計算題  公式參數
leftItems                    陣列         配合題  左欄項目
rightItems                   陣列         配合題  右欄選項（含干擾）
pairs                        陣列         連連看  配對清單
explanation                  物件         建議    解析內容（可由引擎生成）
tips                         物件         建議    答題秘訣（可由引擎生成）
logic                        陣列         建議    邏輯標籤（可由引擎生成）
spec                         物件         建議    規格模板（可由引擎生成）
knowledge_base               物件         建議    知識庫關聯資訊

【五】格式統一檢查清單

製作或修復題庫時，請確認：

必填欄位：
  □ id — 符合 [SMTFCPQL]\d{3} 格式
  □ type — 為 8 大題型之一
  □ typeLabel — 完整的繁體中文名稱（不是空字串）
  □ typePrefix — 正確的單一字母
  □ text — 有內容
  □ answer — 格式符合題型要求
  □ points — 有值
  □ law.pcode — 有效的法規代碼

建議欄位：
  □ stem — 有值（預設「試回答下列問題：」）
  □ group — 有值

多子題題型：
  □ pointsPerItem — 有值（填空、配合、排序、連連看）

題型專用欄位：
  □ blanks — 填空題有值
  □ formulaKey / formulaParams — 計算題有值
  □ leftItems / rightItems — 配合題有值
  □ pairs — 連連看有值
  □ options — 排序題有值（含干擾選項）

不需要的欄位：
  □ 保留 null 或空值（不要刪除）
