🚀 系統網址
-----------
前端：https://occupational-safety-frontend-uju5.onrender.com
後端 API：https://occupational-safety-backend.onrender.com

📚 模式說明
-----------
1. 學科模式
   - 題數：80 題
   - 時間：100 分鐘
   - 題型：單選題（50題）、複選題（20題）、是非題（10題）

2. 術科模式
   - 題數：10 題
   - 時間：120 分鐘
   - 題型：計算題、配合題、排序題、連連看、填空題

3. 計算練習
   - 題數：20 題
   - 時間：60 分鐘
   - 題型：計算題

🛠️ 輔助工具
-----------
1. 計算機（🧮）
   - 科學計算機，支援基本運算
   - 支援：+、-、×、÷、√、²、!、log、ln、exp、π、e
   - 可拖曳移動位置

2. 白板（🎨）
   - 繪圖工具，可做筆記
   - 支援：鉛筆、橡皮擦、顏色選擇、粗細調整
   - 可拖曳移動位置

3. 小鍵盤（⌨️）
   - 數字輸入輔助
   - 支援：數字 0-9、小數點、負號、退格、清除、確定

📊 作答功能
-----------
1. 作答方式
   - 單選題：點選 ○ 圓形選鈕
   - 複選題：點選 ☑ 方形勾選框
   - 是非題：點選 ○ 是 或 ○ 否
   - 填空題：點擊輸入框，使用小鍵盤輸入
   - 計算題：在輸入框輸入答案，可使用計算機輔助
   - 配合題：拖曳下方選項卡片至作答區
   - 排序題：拖曳下方選項卡片至作答區排列
   - 連連看：點選左欄項目 → 點選右欄配對

2. 標記功能（★/☆）
   - 標記題目，方便回頭檢查
   - 點擊「標記★」按鈕即可標記

3. 跳題功能
   - 使用「到第 ___ 題」下拉選單快速跳轉

4. 作答總覽
   - 點擊「作答總覽」按鈕
   - 顯示所有題目的作答狀態
   - 點擊題號可跳轉

📋 解析功能
-----------
點擊「查看所有解析內容」展開解析區：

1. 題幹解析（📖）
   - 題目說明與分析

2. 法源依據（⚖️）
   - 相關法規條文
   - 顯示法規名稱和條文內容

3. 答題秘訣（💡）
   - 核心口訣
   - 關鍵數字
   - 原因邏輯
   - 考試陷阱

4. 邏輯概念（🧠）
   - 題目類型與策略
   - 法規數字、法規記憶、法規理解
   - 危害辨識、計算應用、流程操作

✅ 提交與計分
-----------
1. 提交測驗
   - 點擊「送出」按鈕
   - 系統會提示未作答題數
   - 確認後顯示得分

2. 提前結束
   - 點擊「提前結束測試」按鈕
   - 系統會提示已作答題數
   - 確認後結束測驗

⌨️ 快捷鍵
-----------
- ← 左箭頭：上一題
- → 右箭頭：下一題
- ESC：關閉作答總覽
- G：開啟作答總覽

📱 注意事項
-----------
1. 作答進度會自動儲存（localStorage）
2. 重新整理頁面不會遺失作答記錄
3. 建議使用 Chrome / Edge 瀏覽器
4. 如遇問題，請清除瀏覽器快取
